## Project Overview

This project is a single-page responsive restaurant homepage built using HTML5 and CSS3.  
No templates or external frameworks were used.

The site includes:

- Custom color palette applied using CSS
- Responsive navigation menu (CSS-only hamburger menu, no JavaScript)
- Background hero image using CSS background-size and positioning properties
- 3 optimized images with rounded corners and filter effects
- Google Fonts integration
- Icons for enhanced UI
- Accessibility-friendly semantic HTML structure
- External stylesheet for all CSS
- Validated using W3C Markup Validation Service
- Tested with WAVE Web Accessibility Tool
